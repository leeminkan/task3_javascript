Có những cách đặt Javascript nào trong file HTML? Cho ví dụ minh họa.

Cách 1: Internal - viết luôn trong file html. 

          
            <html>
                <head>
                    <title></title>
                    <script language="javascript">
                        alert("Hello World!");
                    </script>
                </head>
                <body>
                     
                </body>
            </html>

Cách 2: External - viết một file js rồi sau đó import vào 

          
            <script language="javascript" src="demo.js"></script>

Lúc này trong file demo.js sẽ không cần cặp <script></script> nữa, trình duyệt sẽ biết được định dạng của file này là javascript.

Cách 3: Inline - trực tiếp trong thẻ html 

          
            <input type="button" onclick="alert(1)" value="Click Me"/>

Trong đoạn code trên, khi ta click vào button sự kiện onclick sẽ kích hoạt để thực thi lệnh alert(1) - đưa ra popup có giá trị 1

Hiển thị trong một phần tử HTML, sử dụng innerHTML.

Khi đó chúng ta phải lấy được phần tử HTML bằng phương thức document.getElementById(id)

id là để xác định phần tử HTML,còn innerHTML là thuộc tính xác định nội dung bên trong phần tử đó.

          
            <!DOCTYPE html>
            <html>
            <body>

            <h1>My First Web Page</h1>
            <p>My First Paragraph</p>

            <p id="demo"></p>

            <script>
            document.getElementById("demo").innerHTML = 5 + 6;
            </script>

            </body>
            </html>

Kết quả:

My First Web Page

My First Paragraph.

Hiển thị vào file HTML, sử dụng document.write()

Ví dụ:

          
            <!DOCTYPE html>
            <html>
            <body>

            <h1>My First Web Page</h1>
            <p>My first paragraph.</p>

            <script>
            document.write(5 + 6);
            </script>

            </body>
            </html>

Kết quả:

My First Web Page

My first paragraph.

Lưu ý là sử dụng document.write() sau khi trang đã được load sẽ xóa toàn bộ các phần tử tồn tại trên trang đó.

Ví dụ:

          
            <!DOCTYPE html>
            <html>
            <body>

            <h1>My First Web Page</h1>
            <p>My first paragraph.</p>

            <button type="button" onclick="document.write(5 + 6)">Try it</button>

            </body>
            </html>

Kết quả: Khi click vào button
