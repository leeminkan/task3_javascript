Có những cách đặt Javascript nào trong file HTML? Cho ví dụ minh họa.

Cách 1: Internal - viết luôn trong file html. 


	<html>
	<head>
	<title></title>
	<script language="javascript">
	alert("Hello World!");
	</script>
	</head>
	<body>

	</body>
	</html>

Cách 2: External - viết một file js rồi sau đó import vào 


	<script language="javascript" src="demo.js"></script>

Lúc này trong file demo.js sẽ không cần cặp <script></script> nữa, trình duyệt sẽ biết được định dạng của file này là javascript.

Cách 3: Inline - trực tiếp trong thẻ html 


	<input type="button" onclick="alert(1)" value="Click Me"/>

Trong đoạn code trên, khi ta click vào button sự kiện onclick sẽ kích hoạt để thực thi lệnh alert(1) - đưa ra popup có giá trị 1

Trong Javascript, kiểu dữ liệu được chia thành các loại cơ bản sau:

  • String
  • Number
  • Boolean
  • Object
  • Undefined
  • Array

String

Đây là kiểu chuỗi, được gán trong dấu nháy đơn hoặc kép. Trường hợp trong chuỗi cũng có xuất hiện dấu nháy đơn hoặc nháy đôi thì bắt buộc bạn phải thêm ký tự \ đằng trước dấu nháy đó nếu không sẽ bị lỗi cũ pháp.

Lưu ý: Khi bạn muốn Enter xuống hàng một chuỗi trong Javascript thì bắt buộc phải sử dụng dấu + để nối chuỗi nếu không sẽ bị lỗi cú pháp.

Ví dụ cách khai báo:


	<script>
	var a = "Chuoi trong ngoac kep";
	var b = 'Chuoi trong ngoac don';
	var vidu = "Tui tên \"Phúc\"";
	// Đúng
	var xuongdong = "Cách đúng"
	+ "nè";
	// Sai
	var xuongdong = "Cách sai
	nè";
	</script>

Number

Number bao gồm số nguyên với độ chính xác là 15 chữ số, số thực với độ chính xác là 17 chữ số. Ngoài ra, còn có giá trị NaN (Not a Number) đây là giá trị thuộc kiểu Number nhưng không phải là một số. Mặc khác, còn có Infinity là giá trị đại diện cho một số rất lớn, nằm ngoài khả năng đo đếm của Javascript.

Ví dụ về cách khai báo:


	<script>
	var a = 22;
	var b = -1993;
	</script>

Nếu số quá lớn, ta có thể thể hiện nó sang dạng mũ:

	                        			
	                        				<script>
	                        				document.write("5e+9 sẽ bằng với: " + (5e+9));
	                        				</script>

Boolean

Kiểu dữ liệu này chỉ có hai giá trị là true và false. Có 2 cách để nhận được giá trị này là gán trực tiếp và nhận từ một điều kiện.

Ví dụ về cách gán: 


	<script>
	var a = true;
	var b = false;
	var c = 6 > 2;
	</script>

Object

Đây là kiểu dữ liệu đối tượng, trong đó bao gồm các thuộc tính là các kiểu dữ liệu khác nhau cùng với những phương thức tương tự như hàm nhưng chỉ được gọi với đối tượng đó.

Ví dụ về cách gán: 


	<script>
	var SinhVien = {
	name:"Phúc",
	gender:"Nam",
	year:1998
	}
	</script>

Undefined

Đây là kiểu dữ liệu thể hiện rằng biến chưa được xác định, chwua có kiểu dữ liệu cụ thể.

Ví dụ khi khai báo biến mà không gán giá trị thì kiểu dữ liệu là undefined, hoặc ta cũng có thể gán trực tiếp.


	<script>
	var myName; //Lúc này là undifined
	var myName = "Phuc" //Kiểu String
	var myName = undefined;
	</script>

Array

Array là mảng là trường hợp đặc biết cảu Object. Thật chất thì Array có kiểu dữ liệu là Object.

Ví dụ tạo mảng: 


	<script>
	var mang = ["phantu1", "phantu2", "phantu3"];
	</script>

Để xác định kiểu dữ liệu của biến, ta sử dụng toán tử typeof

Ví dụ: 


	<script>
	var b = typeof "Phuc"; //string
	var c = typeof 1998; //number
	var d = typeof true; //boolean
	</script>

Làm thế nào để truy xuất 1 thẻ HTML bất kỳ theo ID, class, Name, tên thẻ HTML và theo CSS Selector? Cho ví dụ minh họa.

Truy xuất theo ID, ta sử dụng phương thức document.getElementById('id')

Ví dụ


	// Lấy thẻ input
	var element = document.getElementById('website');

	// Lấy giá trị của thẻ input
	document.write(element.value);

Truy xuất theo Name, ta sử dụng phương thức document.getElementByName(name)

Ví dụ


	// Lấy thẻ input
	var element = document.getElementByName('website');

Truy xuất theo tên thẻ HTML, ta sử dụng phương thức document.getElementsByTagName('tagname')

Ví dụ


	// Lấy thẻ input
	var element = document.getElementsByTagName('input');

	// Lấy giá trị của thẻ input
	document.write(element[0].value);

Truy xuất theo class, ta sử dụng phương thức document.getElementsByClassName('input')

Ví dụ


	// Lấy thẻ input
	var element = document.getElementsByClassName('website');

	// Lấy giá trị của thẻ input
	document.write(element[0].value);

Truy xuất theo Css Selector, ta sử dụng phương thức document.querySelectorAll("selector.css")

Ví dụ: Giả sử ta có một đoạn mã HTML như sau:


	<html>
	<body>
	<input type="text" value="thẻ không cần lấy" class="website"/>
	<div>
	<input type="text" value="Thẻ Cần Lấy" class="website"/>
	<input type="text" value="thẻ không cần lấy"/>
	</div>
	</body>
	</html>

Và giờ khi muốn truy vấn đến thẻ input có class website nằm trong thẻ div thì ta có cú pháp sau, kết quả trả về là một mảng các Objects:


	var element = document.querySelectorAll("div input.website");
